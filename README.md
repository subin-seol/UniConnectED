# UC-Koala  

Welcome to the **UC-Koala repository**, the student capstone project for **UniConnected** (Semester 2, 2025). This page provides an overview of UniConnected as well as guidance on using this repository.

---

## About UniConnected  

UniConnected is a growing startup that offers a **matching platform** connecting:  
- University students  
- Alumni  
- Academics  
- Industry partners  

### Current Success  
- Successfully piloted at the **University of Melbourne – Faculty of Education** for industry placement  
- **250+ students** actively using the platform  
- **100 schools** onboard as industry partners  
- Ready for **major expansion** across universities  

## About This Repository  

The `./docs` folder contains a **snapshot of the project wiki**. This repository does not include `./src` files, as development happens in the [client’s repository](https://github.com/uniconnected/uniconnected-django)
